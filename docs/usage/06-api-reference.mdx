# API Reference

Maestro's API is a RESTful API that uses JSON for request and response bodies. It follows standard HTTP methods and status codes, providing a robust interface for managing Maestro-related data in the Overture platform.

### Access

Depending on your environment, the Maestro Swagger API can be accessed from the following URLs:

- Development URL: `http://localhost:11235/api-docs`
- Example production URL: `https://maestro.demo.overture.bio/api-docs`
